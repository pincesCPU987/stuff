Calculator

Instructions

Use 'pi' to use the constant pi.

Use 'sqrt(number)' to take square roots.

Use 'power(base,exponent)' to raise numbers to a power.

Use 'sq(number)' to square numbers.

Use 'result' to pull out the last result. This will not work properly if you just loaded the page.

To use scientific notation, use for example: '4e+3' for big numbers and '4e-3' for small numbers. These specific ones will return 4000 and 0.004. The numbers can change to anything needed.

To find a factorial such as 9!, use fact(integer). If the number is not an integer, it won't work.
